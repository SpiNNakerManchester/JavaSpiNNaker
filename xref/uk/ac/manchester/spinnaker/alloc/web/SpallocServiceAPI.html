114 	 * asynchronous manner when it EITHER calls into the waitable epoch system,
115 	 * OR if it is doing a database write AT ALL because acquiring a write lock
116 	 * can take a while with SQLite if things are busy.
117 	 *
118 	 * That is not an exhaustive list of reasons; anything that causes a request
119 	 * to wait for more than a millisecond is also a candidate.
