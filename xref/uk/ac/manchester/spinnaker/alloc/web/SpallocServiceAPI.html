112 	 * When adding to this interface, an operation should be implemented in an
113 	 * asynchronous manner when it EITHER calls into the waitable epoch system,
114 	 * OR if it is doing a database write AT ALL because acquiring a write lock
115 	 * can take a while with SQLite if things are busy.
116 	 *
117 	 * That is not an exhaustive list of reasons; anything that causes a request
118 	 * to wait for more than a millisecond is also a candidate.
